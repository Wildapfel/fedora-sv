## fedora-silverblue

**1. Layer OStree**

- followed by a reboot: `systemctl reboot`
- also installs vscode

```Shell
./fedora_silverblue_system --ostree+vscode
```

**2. Install Tools**

- must follow some instructions:

```bash
./fedora_silverblue_system.sh --tools
```

**3. Install Apps**

```bash
./fedora_silverblue_system.sh --apps
```
